# lab3-patterns
Лабораторная работа №3 - Порождающие паттерны проектирования
## Цель:
Реализация порождающих паттернов проектирования на Python
## Паттерны:
1. Singleton - гарантирует один экземпляр класса
2. Factory Method - создание объектов через подклассы  
3. Abstract Factory - создание семейств связанных объектов
4. Builder - поэтапное создание сложных объектов
